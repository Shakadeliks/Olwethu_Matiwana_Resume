Men's fashion E-commerce website

My first website project

This is the first full website that I ever created, intended as a capstone project for the HyperionDev web development course I took. A major part of the project was to add various functionalities to the site with Javascript. This was intended to demonstrate all of the HTML, CSS and Javacript skills we had learnt so far in the first level of the course.

The site allows for the user to quick add items seen on the Catalogue page, with the click of one button. As items are added to the cart, the cart button the nav bar updates to show how many items are currently in the cart. Once the is ready to chechout, they can click on the nav bar cat button, which will direct them to the checkout. here the user can see what is in their cart, each item's price and the total cost including VAT.

The user can select they would like their order to be delivered or collected. Should the delivery option be chosen, a form will appear in a slide down animated fashion, which will ask for their delivery address. Once the order has been confirmed the user is provided an order number via a Javascript alert that displays the generated order number. The site includes an animated gradient effect on the background on all of the pages and lastly, a contact page with a form for the users detail and their query.
